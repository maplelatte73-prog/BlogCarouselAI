# BlogCarouselAI

## 소개

BlogCarouselAI는 온라인 쇼핑몰 상품 정보와 고객 리뷰를 분석하여 블로그 콘텐츠와 SNS 캐러셀 콘텐츠를 자동 생성하는 AI 도구입니다.

## 주요 기능

* 상품 상세페이지 분석
* 고객 리뷰 분석
* 네이버 블로그 글 생성
* SEO 제목 생성
* 인스타그램 캐러셀 카드뉴스 생성
* 이미지 생성 프롬프트 작성

## 기술 스택

* Python
* OpenAI API
* Streamlit
